My first code editor was TextEdit. Then I learned about this thing called Vim and that was my IDE for many years. When I started teaching web development, I needed to switch to something easier for my students to use, so I switched to Atom. I am now a lead and I dont' code every day, which means I don't use Vim everyday, which means I am rapidly forgetting everything about Vim, except how to exit. My engineers all use VSCode, so this old dev is learning new tricks. 




I love to embed emojis in my Markdown, so this was the first, and most important, package I installed: 

[emojisense](https://marketplace.visualstudio.com/items?itemName=bierner.emojisense)

I write a lot of Markdown files. Stock VSCode accepts suggestions when you press Enter, meaning every line break results in a series of asterisks. **Very annoying!**


```js
 "editor.acceptSuggestionOnEnter": "off" 
```
